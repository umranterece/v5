rs-agora-module - Kapsamlı Adım Adım Dokümantasyon

1. Proje Oluşturma
  1. Yeni bir Vue 3 projesi başlatmak için terminalde şunu çalıştırın:
    npm create vue@latest my-agora-project
  2. Kurulum sırasında aşağıdaki seçimleri öneririz:
    • TypeScript: No
    • JSX Support: No
    • Vue Router: Yes
    • Pinia: Yes
    • Vitest: No
    • End-to-End Testing: No
    • Linting: Oxlint veya ESLint
    • Prettier: Yes
    • Vue DevTools: Yes
  3. Proje klasörüne girin:
    cd my-agora-project
2. Bağımlılıkların Kurulumu
  1. Agora Vue paketini ve gerekli peer dependency'leri yükleyin:
    npm install rs-agora-module
npm install agora-rtc-sdk-ng@^4.0.0 pinia@^3.0.0 mitt@^3.0.0
Not: agora-rtc-sdk-ng, pinia ve mitt peer dependency olarak gereklidir.
3. Stil Dosyasını Dahil Etme
  1. Arayüzün düzgün görünmesi için main.js veya main.ts dosyanıza şu satırı ekleyin:
    import 'rs-agora-module/dist/rs-agora-module.css'
Not: CSS importu zorunludur! Aksi halde arayüz bozuk veya stilsiz görünebilir.
4. Ana Kurulum (main.js)
import { createApp } from 'vue'
import { createPinia } from 'pinia'
import App from './App.vue'
import 'rs-agora-module/dist/rs-agora-module.css'

const app = createApp(App)
app.use(createPinia())
app.mount('#app')
5. App.vue Örneği
<template>
  <div class="app">
    <AgoraConference
      :channelName="'test-channel'"
      :autoJoin="true"
      :userUid="userUid"
      :tokenEndpoint="tokenEndpoint"
      @joined="onJoined"
      @left="onLeft"
      @error="onError"
    />
  </div>
</template>

<script setup>
import { ref } from 'vue'
import { AgoraConference } from 'rs-agora-module'

const userUid = ref(Math.floor(Math.random() * 100000))
const tokenEndpoint = ref('https://your-api.com/createToken.php')

const onJoined = (data) => { console.log('Kanala katıldı:', data) }
const onLeft = (data) => { console.log('Kanaldan ayrıldı:', data) }
const onError = (error) => { console.error('Hata:', error) }
</script>
6. Props ve Eventler
Prop Tip Açıklama
channelName String Agora kanal adı
autoJoin Boolean Otomatik katılım (varsayılan: false)
userUid String/Number Kullanıcı ID (null ise random atanır)
tokenEndpoint String Token API endpoint adresi

Eventler

Event Açıklama
joined Kanala başarıyla katılınca
left Kanaldan ayrılınca
error Hata oluşunca
7. Sıkça Sorulan Sorular & Sorun Giderme
  • CSS neden ayrıca import ediliyor?
    Vue kütüphanelerinde stil dosyaları otomatik yüklenmez, ana projede elle import edilmelidir.
  • Kamera/mikrofon çalışmıyor, neden?
    Tarayıcıdan izin verilmeli ve HTTPS üzerinde çalışmalısınız.
  • Componentler DOM'da var ama arayüz bozuk?
    CSS importunu kontrol edin. Doğru dosya yolunu kullandığınızdan emin olun.
  • Token endpoint nedir?
    Kendi backend'inizde oluşturduğunuz bir API adresidir. Örnek: https://your-api.com/createToken.php
  • Başka bir kullanıcı ile test etmek istiyorum?
    Farklı bir tarayıcı veya cihazdan aynı kanala girerek test edebilirsiniz.
8. Gelişmiş Kullanım ve Özelleştirme
  • Kendi temanız için CSS değişkenlerini override edebilirsiniz.
  • Component event’lerini kullanarak kendi UI’nizi geliştirebilirsiniz.
  • Birden fazla konferans odası veya gelişmiş kullanıcı yönetimi için Vue Router ve Pinia ile entegre çalışabilirsiniz.
